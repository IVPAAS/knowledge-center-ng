layout: page
title: "Are there any content entitlement controls that apply on KMC users?"
date: 2012-06-18 02:13:43
---

Content entitlement controls are set from the KMC but apply initially MediaSpace (or any other developed integrations). Each KMC user with permission to manage content will have access to the entire set of content managed in the account from the KMC.
