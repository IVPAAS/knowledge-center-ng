<p class="mce-procedure">
  To add a category
</p>

1.  Select the Content tab and then select the Categories tab.
2.  Select Add Category.  
    The New Category window is displayed.<img src="{{site.url}}/assets/717">
3.  Select No Parent if you want the category to be at the top level or select the parent category under which the new category should be created and select Next. We recommend that you minimize categories at the top level. The New Category window is displayed.<img src="{{site.url}}/assets/718">
4.  <a href="http://knowledge.kaltura.com/node/640" target="_blank">Add metadata</a> and click Save & Close.
